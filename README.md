# ArticleGen

AI-powered article generation platform with a modern, professional layout designed to resemble major article sites.

## Features

### Layout & Design
- **Modern Article Site Layout**: Clean, professional design inspired by major news and content sites
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Typography**: High-quality Inter font with excellent readability
- **Color Scheme**: Professional blue and gray palette with high contrast
- **Accessibility**: WCAG-compliant with keyboard navigation and screen reader support

### Pages
- **Homepage (`index.html`)**: Features hero section, article grid, sidebar with widgets
- **Article Page (`article.html`)**: Full article layout with structured content, author info, and related articles

### Components
- **Header**: Sticky navigation with logo, menu, and action buttons
- **Hero Section**: Eye-catching gradient background with call-to-action
- **Article Cards**: Hover effects and engaging preview layouts
- **Sidebar**: Newsletter signup, categories, and recent articles
- **Footer**: Comprehensive site links and social media integration

### Interactive Features
- **Mobile Menu**: Responsive navigation for mobile devices
- **Newsletter Signup**: Form validation with success/error notifications
- **Article Sharing**: Social media sharing buttons
- **Smooth Scrolling**: Enhanced user experience
- **Search Integration**: Ready for search functionality
- **Progressive Enhancement**: Works without JavaScript

## Technical Details

- **HTML5**: Semantic markup for better SEO and accessibility
- **CSS3**: Modern styling with Grid and Flexbox layouts
- **Vanilla JavaScript**: No external dependencies for better performance
- **Progressive Web App Ready**: Modern standards compliance
- **Performance Optimized**: Efficient loading and rendering

## Browser Support

- Chrome/Edge (modern versions)
- Firefox (modern versions)
- Safari (modern versions)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Getting Started

1. Open `index.html` in a web browser
2. Navigate to `article.html` to see the article layout
3. Test responsiveness by resizing the browser window
4. Try the interactive features (newsletter signup, mobile menu)

## Customization

The layout uses CSS custom properties and modular styles for easy customization:

- **Colors**: Modify the color scheme in the CSS variables
- **Typography**: Change font families and sizes
- **Layout**: Adjust grid and flexbox properties
- **Content**: Update HTML content and images

## Performance

- **Optimized Images**: Uses responsive image sizing
- **Minimal Dependencies**: Only uses Google Fonts
- **Efficient CSS**: Organized and optimized styling
- **Fast Loading**: Minimal JavaScript for better performance
